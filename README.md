# Diabetes Classification Prediction (Python)

This project builds a binary classification model to predict whether a patient is diabetic based on clinical features. The workflow includes data preprocessing, model training, and performance evaluation using classification metrics.

## Tech Stack
- Python  
- Pandas & NumPy  
- Scikit-learn  
- Matplotlib & Seaborn  

## Key Work
- Data cleaning and exploratory analysis  
- Train-test split and feature scaling  
- Binary classification modeling  
- Confusion matrix visualization  
- Model evaluation using accuracy and classification metrics  

The notebook demonstrates an end-to-end machine learning pipeline from structured healthcare data to predictive modeling.
